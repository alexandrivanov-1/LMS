# Консультации по проекту LMS

Документ описывает формат консультаций в чате без изменений кода.

## Допустимые темы
- Процессы разработки, тестирования и релизов.
- Планирование задач и оценка рисков.
- Архитектура сервисов и взаимодействие компонентов.
- Анализ логов CI/Integration и интерпретация ошибок.

## Темы, требующие согласования
- Изменения API и контрактов.
- Обновления схемы данных и миграции.
- Изменения политики безопасности или доступа.
- Массовые правки документации, влияющие на продуктовые решения.

## Правила общения
1. Вопрос формулируется максимально конкретно.
2. Ответ содержит факты, гипотезы и рекомендации по действиям.
3. Если данных недостаточно — запрашиваем дополнительную информацию.
4. Спорные моменты фиксируем как follow-up задачи или issues.

## Результат консультации
- Краткое резюме обсуждения.
- Список решений или открытых вопросов.
- Ссылки на связанные задачи, PR или документы.
