# Сайт экспертно-аналитического центра «МЕРА»

Премиальный одностраничный сайт для аналитического экспертного центра
(аудиты и комплаенс → стандарты и аттестация → отбор преподавателей
и программ → обучение с измерением результата). Целевая аудитория —
крупные компании и госсектор.

## Как посмотреть
Файл полностью автономный — откройте `index.html` в браузере
(шрифты подтягиваются с Google Fonts; без сети сайт работает
на системных шрифтах).

```bash
# или поднять локальный сервер
python3 -m http.server 8080 --directory site
# → http://localhost:8080
```

## Что внутри
- Один файл `index.html`: разметка, стили и JS без зависимостей и сборки.
- Светлая современная тема (шрифты Fraunces + Inter), премиум-палитра.
- Анимации появления блоков и счётчиков — **только как улучшение**:
  контент полностью виден и без JavaScript (правило скрытия заскоуплено
  под класс `.js`, который навешивается в `<head>`).
- Адаптив: десктоп, планшет, мобильный (бургер-меню с `aria-expanded`).
- Доступность: `:focus-visible`-кольца, контраст текста ≥ WCAG AA,
  `label` у всех полей формы, `prefers-reduced-motion`.
- Форма заявки в демо-режиме (показывает подтверждение без отправки).

## Проверки
В каталоге проверено статически (браузер в окружении недоступен —
заблокирован сетевой политикой): валидность инлайн-JS, баланс тегов,
разрешимость якорей, отсутствие скрытого без JS контента, определённость
CSS-переменных. Дополнительно — независимый ревью на доступность и вёрстку.

## Что заменить перед запуском
- Контакты (телефон, почта, адрес) — сейчас заглушки.
- Цифры в блоках статистики и кейсов — сейчас демонстрационные.
- Имена/описания экспертов.
- Подключить реальный обработчик формы (бэкенд или сервис форм).
