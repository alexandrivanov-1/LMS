# Сайт экспертно-аналитического центра «МЕРА»

Премиальный одностраничный сайт в стиле госорганов и госкорпораций
(аудиты и комплаенс → стандарты и аттестация → отбор преподавателей
и программ → обучение с измерением результата). Целевая аудитория —
крупные компании и госсектор.

## Как посмотреть
Файл полностью автономный — откройте `index.html` в браузере
(шрифты Golos Text + Inter подтягиваются с Google Fonts; без сети
сайт работает на системных шрифтах).

```bash
# или поднять локальный сервер
python3 -m http.server 8080 --directory site
# → http://localhost:8080
```

## Интерактив и «крючки»
- **Экспресс-диагностика** — квиз на 6 вопросов с прогресс-баром,
  кольцевым индикатором результата и подстановкой темы в форму заявки.
  Без JS показывается статический чек-лист самопроверки.
- **Hero-дашборд** — анимированные карточки аналитики (кольцевой
  индекс, падающие столбцы нарушений, живой счётчик реестра)
  с параллаксом за курсором.
- **Бегущая строка нормативов** (44-ФЗ, 223-ФЗ, 273-ФЗ, 115-ФЗ,
  ГОСТ Р ИСО 9001…) — чистый CSS, пауза по наведению.
- **Табы «Бизнес / Госсектор»** — чистый CSS (radio hack), работают без JS.
- **FAQ-аккордеон** — нативные `<details>`, работают без JS.
- **Spotlight за курсором** на карточках, «магнитная» CTA-кнопка,
  прогресс-бар прочтения страницы, плавающая кнопка диагностики.
- **Мобильное меню и «версия для слабовидящих»** — чистый CSS
  (checkbox hack), работают без JS.

## Психология интерфейса (что зашито)
- Социальное доказательство: «сейчас в работе 14 проектов», чипы метрик.
- Дефицит: «свободных окон в квартале: 2», «не более шести проектов».
- Снятие тревоги: блок «что произойдёт после заявки», NDA по умолчанию.
- Ценовой якорь: «проекты от 2,5 млн ₽» в FAQ.
- Низкий порог входа: бесплатная диагностика без ввода контактов.
- Эффект Зейгарник: прогресс-бар квиза стартует заполненным на 8%.

## Инварианты (не ломать)
- Весь смысловой контент виден **без JavaScript**: скрытие reveal-блоков
  заскоуплено под класс `.js`; квиз без JS заменяется чек-листом.
- Меню, табы, FAQ, версия для слабовидящих — на чистом CSS.
- `prefers-reduced-motion` и режим для слабовидящих отключают анимации.

## Что заменить перед запуском
- Контакты (телефон, почта, адрес) — сейчас правдоподобные заглушки.
- «Политику обработки персональных данных» сделать ссылкой на документ.
- Цифры статистики, кейсов и чипов — демонстрационные.
- Названия/номера аккредитаций и лицензий.
- Имена и описания экспертов.
- Подключить реальный обработчик формы (бэкенд или сервис форм).
